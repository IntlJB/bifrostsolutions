# Bifrostsolutions

Professionel hjemmeside for Bifrostsolutions med fokus på billige opstartspriser, webdesign, hosting, drift og kontaktformular.

## Teknisk

- React + Vite
- Formspree kontaktformular
- Vercel deployment
- SEO metadata, sitemap og robots.txt

## Kommandoer

```bash
npm install
npm run dev
npm run build
npm run lint
```
